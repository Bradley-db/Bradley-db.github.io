Jupyter Notebook on Mac快捷键整理

2019-05-01

序言：本文对Jupyter Notebook在Mac OSX 的常用快捷键进行简单整理

便于速查，提升工作效率。采用默认Jupyter快捷键配置，如有需要可以根据个人喜好进行对应修改。

1.基础快捷键说明：

基本符号与指令对应关系
⌘: Command
⌥: Option
⇧: Shift
↩: Return
␣: Space
⇥: Tab
